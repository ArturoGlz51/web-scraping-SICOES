<h1>Hola! Te enseñare a utilizar este Script de WebScraping.</h1>
<p>Primero, necesitarás Selenium, lo puedes instalar desde la terminal de Visual Studio Code (CTRL + Ñ) con este comando.</p>
<p>pip install selenium</p>
<br>
<p>Ahora, necesitarás los Requests de Selenium y BeautifulSoup4 Pandas, igual puedes instalarlos desde la terminal.</p>
<p>pip install requests selenium beautifulsoup4 pandas openpyxl</p>
<br>
<p>Por ultimo, necesitarás el WebDriver de Chrome, lo puedes encontrar en la pagina oficial!</p>
<p>https://developer.chrome.com/docs/chromedriver/downloads?hl=es-419</p>
<br>
<h1>Y listo! Puedes comenzar a utilizar nuestro Script para Scraping Web!</h1>
